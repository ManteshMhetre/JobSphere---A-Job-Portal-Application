# NaukriWala Frontend

React.js frontend for the NaukriWala job portal application.

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Features

- Job searching and filtering
- User registration and login
- Profile management
- Job applications
- Employer dashboard
- Resume upload

## Built with

- React.js
- Vite
- React Router
- Axios for API calls

For complete setup instructions, see the main [README.md](../README.md) in the root directory.
